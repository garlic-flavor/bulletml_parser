これは BulletML (http://www.asahi-net.or.jp/~cs8k-cyu/bulletml/) ファイルを読み込むためのプログラムです。

  • 本家アプレットと挙動がなんかちがう。

  • BulletML は ABA Games さんが作ったものです。
    ABA Games (http://www.asahi-net.or.jp/~cs8k-cyu/)
    BulletML (http://www.asahi-net.or.jp/~cs8k-cyu/bulletml/)
  • 2ch D言語 part29 574 さんのゲーム作る流れが、モチベーションです。
    D言語でアクションゲームでも作ってみる？(http://sourceforge.jp/projects/d-action/wiki/FrontPage)
  • D言語用です。
    D Programing Language 2.0 (http://dlang.org/index.html)
  • C言語ライブラリのポーティングに Derelict3 を使っています。
    Derelict (http://dsource.org/projects/derelict)
    Derelict3 (https://github.com/aldacron/Derelict3)
  • SDL2+OpenGL4.2 環境を想定しています。
    SDL (http://www.libsdl.org/)
    OpenGL (http://www.opengl.org/)
  • ドキュメントに JQuery を利用しています。
    JQuery (http://jquery.com/)
    JQuery UI (http://jqueryui.com/)
    Google CDN (https://developers.google.com/speed/)
